# MCP Server

Microservice to manage to IA MCP Interations.

## Setup

```bash
bun install
bun prepare
```

## Environment Variables

Configure the following variables before starting the server:

- `AUTHENTIK_URL`
- `AUTHENTIK_USERNAME`
- `AUTHENTIK_PASSWORD`
- `HOME_ASSISTANT_PLUS_SERVICE_ENDPOINT`
- `HOME_ASSISTANT_PLUS_CLIENT_ID`
- `GOOGLE_SERVICE_ENDPOINT`
- `GOOGLE_SERVICE_CLIENT_ID`
- `DISCORD_SERVICE_ENDPOINT`
- `DISCORD_SERVICE_TOKEN`
